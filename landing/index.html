Features

Everything you need,
nothing you don't.

A complete code-push platform that runs entirely on your infrastructure. No vendor lock-in, no data leaks.

Self-Hosted

Run on your own servers. Your code, your data, your rules. Full control over infrastructure and compliance.

Secure by Default

Patches are signed and verified. SHA-256 checksums ensure integrity. End-to-end encryption optional.

Lightning Fast

Patch checks complete in <50ms. Binary diffs are tiny — most patches are under 100KB.

Phased Rollouts

Roll out patches to 1%, 10%, 50%, or 100% of users. Roll back instantly if something goes wrong.

CLI & API

Full command-line interface and REST API. Integrate with your CI/CD pipeline in minutes.

One-Click Deploy

Docker Compose gets you running in 60 seconds. Kubernetes Helm chart for production workloads.

How it works

Three steps to
freedom from app stores.

01

Deploy the server

Run docker compose up and your patch server is live. Connect it to your domain, add TLS, and you're done.

$ docker compose up -d
02

Patch the Shorebird repos

Our script rewrites all API endpoints to point to your server. Build the CLI and updater — takes about 5 minutes.

$ python patch_repos.py
03

Ship patches instantly

Use the familiar Shorebird CLI to release patches. Your users get updates on next app launch — no store review needed.

$ shorebird release patch
